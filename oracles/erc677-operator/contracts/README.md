# Mina zkApp: erc677 operator

This template uses TypeScript.

## How to build

```sh
pnpm run build
```

## How to run tests

```sh
pnpm run test
pnpm run testw # watch mode
```

## How to run coverage

```sh
pnpm run coverage
```

## FYI

```sh
pnpm install husky -D

```

# Mina zkApp: TicTacToe on Lightnet

This template uses TypeScript.

## How to build

```sh
pnpm run build
```

## How to run tests

```sh
pnpm run test
pnpm run testw # watch mode
```

## How to run coverage

```sh
pnpm run coverage
```

## How to run 'interact.js'

```sh
pnpm run lightnet:up
pnpm run build

node build/src/interact.js

pnpm run lightnet:down

```
